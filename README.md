# 🚀 CoachMind – Final Project

A full-stack web application built for managing basketball players and tracking performance statistics.

---

## 🚀 Features

- User authentication (Firebase)
- Player management system
- Statistics tracking (points, assists, rebounds, etc.)
- Dynamic UI with JavaScript
- MySQL database integration

---

## 🛠️ Technologies Used

- PHP (XAMPP)
- MySQL
- JavaScript
- Firebase Authentication
- HTML & CSS

---

## ▶️ How to Run the Project

1. Install XAMPP on your computer  
2. Place the project folder inside `htdocs`
